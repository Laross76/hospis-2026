# Дипломный проект по профессии «Инженер по тестированию».
## Запуск авто-тестов (условия):
- Android studio
- Java 17
- Эмулятор Android с версией API 29 или выше

## Запуск авто-тестов:
- В Android studio откройте проект
- Откройте тесты
- Кликните на тесты два раза левой кнопкой мыши и запустите тесты кнопкой Run Test или кликните на тесты правой кнопкой мыши и выберите из откывающегося списка Run "Abaut Test"
